Validation logic is implemented in the application to ensure that only correct and meaningful data is processed.

🔹 Implemented Validation Rules
1️⃣ Login Validation
Checks whether username and password are correct
Prevents unauthorized access

2️⃣ Email Input Validation
Subject must not be empty
Message body must not be empty
Ensures the system processes only valid emails

3️⃣ Role Validation
Only valid roles are allowed:
Customer
Admin
Support Executive
Prevents invalid role selection

4️⃣ Data Format Validation
Input must be in proper text format (string)
Avoids incorrect or corrupted data

5️⃣ Basic Security Validation
Filters unwanted or harmful inputs
Prevents invalid characters or scripts
