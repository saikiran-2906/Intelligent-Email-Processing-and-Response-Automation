## Level 0 Diagram – System Overview

The Level 0 diagram provides a high-level overview of the **Intelligent Email Processing and Response Automation System**. It illustrates how the system interacts with external users and handles information flow.

- Customers send emails to the system.
- The system automatically analyzes incoming emails and sends appropriate automated responses.
- The Admin manages keywords, response templates, and system configurations.
- Support Executives receive emails that require human intervention.
- The system maintains detailed email logs for monitoring and tracking purposes.

In short, this diagram represents the **big picture** — who interacts with the system and how information moves between users and the system.

## Level 1 Diagram – Internal Process Flow

The Level 1 diagram breaks down the system into its core internal processes and data handling mechanisms.

### Core Processes

- **Receive Email** – The system accepts incoming customer emails.
- **Preprocess Email** – The email content is cleaned and prepared for analysis.
- **Classify Email** – The system identifies the email category using predefined keywords.
- **Generate Response** – A suitable response template is selected and a reply is generated.
- **Send / Forward Email** – The system sends an automated response or forwards the email to a support executive if manual handling is required.
- **Log Email Details** – Email information is stored for future reference and analysis.

### Data Stores

- **Email Log Database** – Stores the complete history of received and sent emails.
- **Keyword & Template Store** – Contains keywords and response templates used for classification and automated replies.

The Admin can manage templates, keywords, and review email logs through the system interface.
